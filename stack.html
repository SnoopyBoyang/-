堆栈基础实验

  在计算机领域，堆栈是一个不容忽视的概念，但是很多人，甚至是计算机专业的人也没有明确：堆栈其实是两种数据结构。堆栈都是一种数据项按序排列的数据结构，只能在一端（栈顶top）对数据项进行插入和删除。

实验目的

  1.掌握栈的概念和两种存储结构，即顺序存储结构和链式存储结构；
  2.掌握栈的特点，即后进先出的原则；
  3.掌握栈的基本操作，如入栈和出栈等；
  4.运用栈解决实际问题。

实验要求

  通过上机调试理解和掌握栈的含义及基本算法；在此基础上根据设计性实验提示补充程序关键部分使程序满足设定的功能要求。

背景知识

一、栈的概念

             

                图1　栈的示意图

  栈是限定仅在表尾进行插入和删除运算的线性表。表尾端称为栈顶，另一端称为栈底。
  由于只能在栈顶进行插入和删除运算，所以总是后进栈的元素先出来，即后进先出（LIFO）。
二、栈的两种存储结构：顺序栈和链栈
1.顺序栈
  顺序栈是用一组地址连续的存储单元依次存放从栈底到栈顶的若干数据元素。
2.链栈
  链栈是运算受限的单链表，即只能在表头位置进行插入和删除操作。链栈中每个数据元素用一个结点表示，栈顶指针作为链栈的头指针。
三、栈的基本操作实现算法
  因为顺序栈一般都使用数组来存储，属于静态分配，因此实现顺序栈的进栈时，将当前计数值+1，再将要进栈的元素赋给计数值对应的数组元素。除此之外，进栈之前需要判定栈是否已满。
  在实现顺序栈的出栈时，先判定栈是否为空，然后将栈顶元素返回，将计数值-1即可。
  因为链栈中的结点是动态产生的，不需要考虑上溢出问题，所以在实现链栈的进栈时，新建一个结点，将新结点与链栈相连接即可。
  在实现链栈的出栈时，先判空，然后将栈顶指针指向的结点值取出并释放结点空间即可。

上机实验涉及到的工具

C language

  C语言是一种计算机程序设计语言，它既具有高级语言的特点，又具有汇编语言的特点。1978年后，C语言已 先后被移植到大、中、小及微型机上，它可以作为工作系统设计语言，编写系统应用程序，也可以作为应用程序设计语言，编写不依赖计算机硬件的应用程序。它的应用范围广泛，具备很强的数据处理能力。

Java language

  Java是一种简单的，跨平台的，面向对象的，分布式的，解释的，健壮的，安全的，结构的，中立的，可移植的，性能很优异的，多线程的，动态的（我的老天~）语言。当1995年SUN推出Java语言之后，全世界的目光都被这个神奇的语言所吸引。

MySQL data base

  MySQL(通常被读作My-SEQuel)是一个开放源码的关联式数据库管理系统。原开发者为瑞典的MySQL AB公司，该公司在2008年被SUN Microsystem收购。甲骨文公司(Oracle)2009年收购SUN公司，MySQL从此成为Oracke旗下产品。MySQL被广泛地应用在Internet上的中小型网站中。由于其体积小、速度快、总体拥有成本低，尤其是开放源码这一特点，许多中小型网站为了降低网站总体拥有成本而选择了MySQL作为网站数据库。MySQL所使用的SQL语言是用于访问数据库的最常用标准化语言。
