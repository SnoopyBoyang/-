数据库技术——MySQL语言实验

  MySQL是一个开放源码的关联式数据库管理系统。原开发者为瑞典的MySQL AB公司，该公司在2008年被Sun Microsystems收购。甲骨文公司(Oracle)2009年收购Sun Microsystems公司，MySQL成为Oracle旗下产品。MySQL被广泛地应用在Internet上的中小型网站中。由于其体积小、速度快、总体拥有成本低，尤其是开放源码这一特点，许多中小型网站为了降低网站总体拥有成本而选择了MySQL作为网站数据库。MySQL所使用的SQL语言是用于访问数据库的最常用标准化语言。

实验目的

  1.熟练掌握数据库的建立、查看、打开、修改和删除等基本操作方法；
  2.学会利用MySQL进行数据库管理的方法。

实验要求

  在已安装MySQL的计算机上实现数据库的创建、表的创建以及常用的查询操作，并将相应过程截图。

实验内容

  1.利用CREATE DATABASE建立学生信息数据库STU_INFO;
  2.利用SHOW DATABASE查看已有数据库信息;
  3.利用USE，打开指定数据库STU_INFO;
  4.利用CREATE TABLE建立学生表Student，课程表Course:
    Student结构:
    ————代码:No,Name,Sex,Birth
    ————含义:学号，姓名，性别，生日
    ————类型:CHAR,CHAR,CHAR,DATE
    ————宽度:6,8,5,默认
    Course结构:
    ————代码:No,Name,Hour,Credit
    ————含义:课号,课名,学时,学分
    ————类型:CHAR,CHAR,CHAR,CHAR
    ————宽度:4,16,1,1
然后输入至少3个学生的信息在Student里面，输入至少3科在Course里面。
  5.由于学校政策要求，学生表中还需要增加Age这一项，先ALTER后再用ADD增加，并更新每个学生的Age;
  6.由于学生表中最上面的学生跳级，其学号将改变，改变为1901，使用UPDATE实现;
  7.学生表中最后一名学生，由于成绩不好退学，将其从表中删除。

参考实现

  ——进入MySQL
  mysql -u root -p
  root
  ——创建学生信息数据库
  CREATE DATABASE STU_INFO;
  ——打开学生信息数据库
  USE STU_INFO;
  ——建立表Student
  CREATE TABLE Student(
   No CHAR(6),
   Name CHAR(8),
   Sex CHAR(5),
   Birth DATE);
  INSERT INTO Student values('100001','Kevin','man','1994-10-14');
  INSERT INTO Student values('100002','Luffy','man','1994-6-1');
  INSERT INTO Student values('100003','Lily','woman','1995-3-12');
  INSERT INTO Course values('1111','Math','48','4');
  INSERT INTO Course values('2222','English','12','1');
  INSERT INTO Course values('3333','Cplusplus','8','2');
  ALTER TABLE Student add Age char(4);
  UPDATE Student set Age='20' where Name='Kevin';
  UPDATE Student set Age='20' where Name='Luffy';
  UPDATE Student set Age='20' where Name='Lily';
  UPDATE Student set No='1901' where No='100001';
  DELETE from Student where No='100003';

实验报告要求

  输入的学生信息应当具有个性化,不应当出现雷同,对表操作的每一步都应该将其操作后的结果显示出来并截图。

C language

  C语言是一种计算机程序设计语言，它既具有高级语言的特点，又具有汇编语言的特点。1978年后，C语言已 先后被移植到大、中、小及微型机上，它可以作为工作系统设计语言，编写系统应用程序，也可以作为应用程序设计语言，编写不依赖计算机硬件的应用程序。它的应用范围广泛，具备很强的数据处理能力。

Java language

  Java是一种简单的，跨平台的，面向对象的，分布式的，解释的，健壮的，安全的，结构的，中立的，可移植的，性能很优异的，多线程的，动态的（我的老天~）语言。当1995年SUN推出Java语言之后，全世界的目光都被这个神奇的语言所吸引。
