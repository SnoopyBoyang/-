C语言实验部分（学生版）

    增加C语言基础知识实验：主要包括指针和结构体部分。

  • Project 1 指针及其典型应用

        1.理解指针的含义，掌握指针变量的使用方法；
        2.掌握指向数组和指向函数的指针变量的使用方法。
        3.通过上机调试理解和掌握指针的含义及基本使用方法；在此基础上根据设计性实验提示补充程序关键部分使 程序满足设定的功能要求。

    Read more

  • Project 2 结构体及其典型应用

        1.掌握结构体类型变量的定义及使用；
        2.掌握结构体变量的引用形式，结构体数组的应用；
        3.理解结构体作为不同数据类型的一个整体在实际编程中的应用。

    Read more

  • Project 3 二叉树操作

        1.掌握二叉树的定义、性质及存储方式，各种遍历算法；
        2.采用二叉树链表作为存储结构，完成二叉树的建立，先序、中序和后序以及按层次遍历的操作，求所有叶子及结点总数的操作。

    Read more

  • Project 4 图的遍历

        1.掌握有向图和无向图的概念；
        2.掌握邻接矩阵和邻接链表建立图的存储结构；
        3.掌握DFS及BFS对图的遍历操作；
        4.了解图结构在人工智能、工程等领域的广泛应用。

    Read more

  • Project 5 线性表和队列的应用

        1.理解线性表和队列的基本操作；
        2.理解队列的基本算法以及队列的典型应用的原理；
        3.通过上机调试理解和掌握线性表和队列的基本操作；
        4.根据提示补充程序的关键成分代码。
