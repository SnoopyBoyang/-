Eclipse platform install

    Eclipse是一个开放源代码、基于Java的可扩展开发平台。安装Eclipse前请参考“资源下载”中有关文档进行安装和配置JDK。
    C/C++ Development Toolkit(CDT)并不是一个软件，而是基于Eclipse的C/C++开发套件。
    GNU Compiler Collection(GCC)是跨平台的C/C++编译器集合。
    以上软件都是不用担心私有许可和收费问题的。

下载说明

    提供的压缩包中已经包含有Eclipse Indigo、CDT、GCC4.9。
    最新版Eclipse可到Eclipse 下载页面下载。而对应版本的CDT可到此处下载。需要最新版GCC的可移步这里(可能需要FQ，你懂的~)。

Eclipse Indigo安装

    Eclipse的安装非常简单，解开压缩包即可使用。接下来以Eclipse Indigo为例介绍Eclipse的安装与配置。
    打开Eclipse压缩包;
    将Eclipse提取到你喜欢的地方;
    解压完毕就可以到Eclipse目录下双击Eclipse.exe运行了！
    注意：首次运行需要设置工作目录，点击“Brouwse...”选择工作目录，或者直接修改，建议选择一个短的路径！！

GCC4.9编译环境安装

    1.同Eclipse，将64位压缩包(或者是32位压缩包)解压到任意位置;
    2.同Java，需要设置bin路径环境变量;
    3.进行测试！

C language

    C语言是一种计算机程序设计语言，它既具有高级语言的特点，又具有汇编语言的特点。1978年后，C语言已 先后被移植到大、中、小及微型机上，它可以作为工作系统设计语言，编写系统应用程序，也可以作为应用程序设计语言，编写不依赖计算机硬件的应用程序。它的应用范围广泛，具备很强的数据处理能力。

Java language

    Java是一种简单的，跨平台的，面向对象的，分布式的，解释的，健壮的，安全的，结构的，中立的，可移植的，性能很优异的，多线程的，动态的（我的老天~）语言。当1995年SUN推出Java语言之后，全世界的目光都被这个神奇的语言所吸引。

MySQL data base

    MySQL(通常被读作My-SEQuel)是一个开放源码的关联式数据库管理系统。原开发者为瑞典的MySQL AB公司，该公司在2008年被SUN Microsystem收购。甲骨文公司(Oracle)2009年收购SUN公司，MySQL从此成为Oracke旗下产品。MySQL被广泛地应用在Internet上的中小型网站中。由于其体积小、速度快、总体拥有成本低，尤其是开放源码这一特点，许多中小型网站为了降低网站总体拥有成本而选择了MySQL作为网站数据库。MySQL所使用的SQL语言是用于访问数据库的最常用标准化语言。
