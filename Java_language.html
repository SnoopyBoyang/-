Java language

  Java是Java程序设计语言和Java平台(JavaEE,JavaME,JavaSE)的总称。Java Development Kit(JDK)是Java开发工具，Java Runtime Environment(JRE)用于执行Java程序。此次上机实验，我们将使用JDK这一开发工具，进行Java的单链表及二叉树的操作。

实验目的

  1.掌握线性表的链式存储结构的表示和实现方法；
  2.掌握链表基本操作的算法实现;
  3.了解和掌握线性表的逻辑结构和链式存储结构，掌握关于二叉树的基本操作，深刻理解二叉树的三种遍历的算法思想。

实验要求

  实验一：单链表的基本操作
  编程实现如下功能：
  1.创建一个具有指定数据元素的单链表，再用头插法或尾插法添加若干元素，并输出单链表中各元素值，观察输入的内容与输出的内容是否一致;
  2.返回当前链表的长度;
  3.在指定索引插入指定节点，并输出插入后的单链表中各元素值;
  4.获取指定节点的位置，如果获取成功，则显示元素的索引，否则显示该元素不存在;
  5.删除链式线性表中指定索引处的元素，返回被删除的元素，并输出删除后的单链表中各元素值;
  6.删除链表中最后一个元素，并输出删除后的单链表中各元素值;
  7.获取指定索引处的节点，如果获取成功，则显示该元素的值，否则显示该元素不存在。
  实验二：二叉树的基本操作
  根据实验要求，建立一个指定的二叉树，对建立好的二叉树实现插入、删除、查找、遍历等基本操作。

实验内容

  实验一：单链表的基本操作
  1.建立单链表，并在单链表上实现插入、删除和查找等基本操作;
  2.建立双向链表，并在双向链表上实现插入、删除和查找操作(选作实验);
  实验二：二叉树的基本操作
  创建一个根节点指向指定元素的二叉树，插入若干左右子树，对其进行如下基本操作：
  1.返回该二叉树的根节点;
  2.返回该二叉树的深度;
  3.获取该二叉树的指定节点的父节点;
  4.获取该二叉树的指定节点的左子节点;
  5.获取该二叉树的指定节点的右子节点;
  对该二叉树进行前序、中序及后序三种方式进行遍历，并返回遍历结果。

C language

  C语言是一种计算机程序设计语言，它既具有高级语言的特点，又具有汇编语言的特点。1978年后，C语言已 先后被移植到大、中、小及微型机上，它可以作为工作系统设计语言，编写系统应用程序，也可以作为应用程序设计语言，编写不依赖计算机硬件的应用程序。它的应用范围广泛，具备很强的数据处理能力。

Java language

  Java是一种简单的，跨平台的，面向对象的，分布式的，解释的，健壮的，安全的，结构的，中立的，可移植的，性能很优异的，多线程的，动态的（我的老天~）语言。当1995年SUN推出Java语言之后，全世界的目光都被这个神奇的语言所吸引。

MySQL data base

  MySQL(通常被读作My-SEQuel)是一个开放源码的关联式数据库管理系统。原开发者为瑞典的MySQL AB公司，该公司在2008年被SUN Microsystem收购。甲骨文公司(Oracle)2009年收购SUN公司，MySQL从此成为Oracke旗下产品。MySQL被广泛地应用在Internet上的中小型网站中。由于其体积小、速度快、总体拥有成本低，尤其是开放源码这一特点，许多中小型网站为了降低网站总体拥有成本而选择了MySQL作为网站数据库。MySQL所使用的SQL语言是用于访问数据库的最常用标准化语言。
